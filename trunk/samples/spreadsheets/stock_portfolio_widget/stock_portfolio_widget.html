Manage your personalized stock portfolio in an OS X Dashboard Widget using Google Spreadsheets

This sample demonstrates how to use the Google data JavaScript client library in the Mac OS X Dashboard Widget environment. This sample works with a Google Spreadsheet which you create in your account. The Widget then uses this spreadsheet to maintain your stock portfolio and uses the GoogleFinance() function in Google Spreadsheets.

Download and install the Gadget/Widget:

  • OS X Dashboard Widget

Create a Google Spreadsheet:

Visit Google Spreadsheets and create a new spreadsheet. The first row of the spreadsheet should contain the following data. This data represents the column headers. You do not need to enter in any other information into your spreadsheet.

Ticker Price High Low Data Delay Change Change Percentage Shares Total

Manage your Portfolio:

By default, both applications display the stock ticker using stocks available from a public spreadsheet we have defined. If you want to maintain your own portfolio, please go to the preferences for the application and enter information about the spreadsheet that you've created as well as the username and password for your account.
