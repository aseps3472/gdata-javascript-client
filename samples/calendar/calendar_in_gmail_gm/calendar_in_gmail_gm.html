Sample Greasemonkey script to view Google Calendar in Gmail using the GData JS Client Library

This sample demonstrates displaying a list of upcoming calendar events from a Google Calendar in the Gmail interface using a Greasemonkey script.

Greasemonkey is a Firefox plugin used to modify the content of a webpage using user-defined scripts.
